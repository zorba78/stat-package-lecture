모의실험: 기댓값 추정 2

다음과 같은 확률 과정을 고려해 보자

  1. 0과 1 사이 실수 \(x_1\)을 뽑는다.
  2. 연속적으로 0과 1 사이 실수 \(x_2\)를 뽑은 후 \(x_1 + x_2\) 계산
  3. 만약 \(x_1 + x_2 > 1\) 이면 중지 \(\rightarrow\) \(x_1 + x_2 <= 1\) 이면 다시 0에서 1 사이의 실수 \(x_3\)을 뽑은 후 \(x_1 + x_2 + x3\) 계산
  4. 만약 \(x_1 + x_2 + x_3 > 1\) 이면 중지 \(\rightarrow\) \(x_1 + x_2 + x_3 <= 1\) 이면 다시 0에서 1 사이 실수 \(x_4\) 뽑은 후 \(\sum_{i=1}^{4} x_i\) 계산 \(\rightarrow\) \(\sum_{i = 1}^{\infty} x_i > 1\) 일 때 까지 반복

위 과정에서 관심 사항은 0에서 1 사이에서 뽑은 난수들의 합이 1이 넘을 때 까지 추출한 난수의 개수이다. 따라서 확률변수 \(X = 2, 3, 4, \ldots\)로 정의할 수 있다.

hit_norm <- function(ntry) {
 X <- 0
 for (i in 1:ntry) {
   uxy <- runif(2, 0, 1)  
   if(uxy[2] < dnorm(uxy[1])) X <- X + 1
 }
 X/ntry
}

모의실험: 신뢰구간

확률표본 \(X_1, \ldots, X_n \stackrel{i.i.d}{\sim} \mathcal{N}(\mu, \sigma^2)\) (\(n \leq 2\)) 이고 \(S^2\) 가 표본 분산일 때, \[V = \frac{(n - 1)S^2}{\sigma^2} \sim \chi^2_{\nu = n - 1} \] 이고, 이를 이용한 \(\sigma^2\) 의 \((1 - \alpha) \times 100\) % 신뢰구간은 \([0, (n - 1)S^2/\chi^2_{\alpha}]\) 로 나타낼 수 있다.
A. 확률표본 \(X_1,\ldots,X_n\) 의 모분산 \(\sigma^2\) 에 대한 \((1 - \alpha) \times 100\) % 신뢰구간을 구하기 위한 함수 calCI()를 작성 하시오.
# 함수를 작성하시오.

calCI <- function(n = 20, mean = 0, 
                  sigma = 1, alpha = 0.05) {
  x <- rnorm(n, mean = mean, sd = sigma)
  (n - 1) * var(x) / qchisq(alpha, df = n - 1)
}

Newton-Raphson 알고리즘 응용: Gradient descent 알고리즘

Gradient decent (GD) 알고리즘은 목적함수의 일차 미분(기울기)을 이용해 일정한 단계로 일차 미분과 반대 방향으로 값이 작아지는 방향으로 이동하면서 최솟값을 찾는 알고리즘이다. 수업시간에 배운 Newton-Raphson 알고리즘과 유사하지만 2차 미분을 할 필요가 없고 간단히 적용할 수 있기 때문에 최적화 문제에서 범용적으로 사용되고 있는 알고리즘 중 하나이다. GD 알고리즘의 pseudo-code는 다음과 같다.
  1. 초기값(initial value)을 설정한다
  2. \(x_{n+1} = x_n - \lambda_n \nabla f(x_n), ~ \forall n=1,2,\cdots\) 계산
  3. \(||f(x_{n+1}) - f(x_{n})|| < \epsilon\) 이면 반복 중지.
GD 알고리즘을 단순선형회귀모형의 최적 계수를 구하는 문제에 적용하면 잔차제곱합을 최소화 하는 회귀계수를 찾는 문제로 간주할 수 있고, 목적함수는 다음과 같다. \[ \mathcal{L}(\boldsymbol{\mathbf{\beta}}) = (\mathbf{y}_{n \times 1} - \mathbf{X}_{n \times p}\boldsymbol{\mathbf{\beta}}_{p \times 1})^T (\mathbf{y}_{n \times 1} - \mathbf{X}_{n \times p}\boldsymbol{\mathbf{\beta}}_{p \times 1})\] 위 목적함수의 gradient 함수는 과제 3에서와 동일하게 \[\nabla \mathcal{L}(\boldsymbol{\mathbf{\beta}}) = \frac{\partial \mathcal{L}(\boldsymbol{\mathbf{\beta}})}{\partial \boldsymbol{\mathbf{\beta}}} = \mathbf{X}^T\mathbf{X}\boldsymbol{\mathbf{\beta}} - \mathbf{X}^T\mathbf{y}\] 로 나타낼 수 있다.
  1. \(\boldsymbol{\mathbf{\beta}}_{old}\) 초기값을 설정한다.
  2. \(\mathcal{L}(\boldsymbol{\mathbf{\beta}}_{old})\) 계산
  3. \(\boldsymbol{\mathbf{\beta}}_{new} := \boldsymbol{\mathbf{\beta}}_{old} - \lambda \nabla \mathcal{L}(\boldsymbol{\mathbf{\beta}}_{old})\)를 계산
  4. \(\mathcal{L}(\boldsymbol{\mathbf{\beta}}_{new})\) 계산
  5. \(|\mathcal{L}(\boldsymbol{\mathbf{\beta}}_{old}) - \mathcal{L}(\boldsymbol{\mathbf{\beta}}_{new})|\) 가 설정한 임계치(tolerance)보다 작아질 때 까지 반복

함수 인수 초기값

  • max_iters=50000
  • lambda = 0.00001
  • bvec = c(0, 0)
  • tol = 1e-7
# 함수를 작성 하시오
- ∇L(B)가 grad_fun()임
- 과제3 N-R 알고리즘과 유사하지만 반복 종료 조건으로 목적함수 간 차이에 대한 설정이 필요함
# obj_fun(): 과제 3 answer와 동일
obj_fun <- function(x, y, beta) {
  n <- length(x)
  xmat <- cbind(1, x)
  xb.vec <- drop(xmat %*% beta)
  sum((y - xb.vec)^2)
}
# grad_fun(): 과제 3 answer와 동일
grad_fun <- function(x, y, beta) {
  n <- length(x)
  xmat <- cbind(1, x)
  xb.vec <- drop(xmat %*% beta)
  (t(xmat) %*% xb.vec - t(xmat) %*% y)
}
gd_linreg <- function(x, y, 
                      max_iters = 50000, 
                      bvec = c(0, 0), 
                      lambda = 0.0001, 
                      tol = 1e-7) {
  iters <- 0
  # loss_init <- obj_fun(x, y, bvec)
  converge = FALSE
  while(iters < max_iters & converge == FALSE) {
    gvec <- grad_fun(x, y, bvec)
    loss_ob <- obj_fun(x, y, bvec)
    bnew <- bvec - lambda * gvec
    loss_nb <- obj_fun(x, y, bnew)
    iters <- iters + 1
    bvec <- bnew
    if (abs(loss_ob - loss_nb) < tol) converge = TRUE
  }
  cat(iters, " iteration.\n")
  return(bvec)
}
